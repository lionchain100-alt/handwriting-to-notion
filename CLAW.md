# AI Agent Rules (OpenClaw)

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Prisma

## Project Structure Rules
- All features must be created inside `/features`
- API routes go in `/app/api`
- Shared logic goes in `/lib`
- Do NOT duplicate code
- Always reuse existing components

## Coding Rules
- Use TypeScript strictly (no any)
- Use functional components only
- Use async/await (no .then)
- Keep functions small and modular

## Task Execution Rules
When implementing a feature:
1. Create types first
2. Create service logic
3. Create API route
4. Create UI

## Forbidden
- Do not modify unrelated files
- Do not install unnecessary dependencies
- Do not rewrite existing working code

## Output Style
- Return complete code
- Include file paths
- Explain only when necessary
